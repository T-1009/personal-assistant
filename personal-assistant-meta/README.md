# Personal Assistant Meta

系统的 **design hub**，所有设计讨论、架构决策和变更规划在此目录下进行。

## 目录结构

```
personal-assistant-meta/
├── specs/          # 功能规格 — 用户视角的系统能力描述
├── architecture/   # 系统架构 — 技术视角的组件与协作设计
├── issues/         # 变更任务 — features / bugs / refactor / chores
├── AGENTS.md       # Meta 域专用 instructions
└── README.md       # 本文件
```

## 快速导航

| 入口 | 内容 |
|------|------|
| `specs/overall_specifications.md` | 总体功能规格书 |
| `architecture/overall_architecture.md` | 总体架构设计 |
| `issues/` | 所有变更 issue 和 Implementation Plan |
| `architecture/ADR/` | Architecture Decision Records |
